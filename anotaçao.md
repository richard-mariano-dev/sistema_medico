# PRINCIPAIS COMANDOS DO GIT 
## Enviando a primeira versao
1. GIT INIT -> inicializar a pasta como repositório local
2. git branch -M main -> Altera a brach master para main
3. git add . -> Adicionar or arquivos para o repositório local
4. git commit -m "Primeira versao do sistema"
5. git remote add origin https://github.com/richard-mariano-dev/sistema_medico.git
6. git push -u origin main

## Enviando as próxinmas versões
1. git add . -> Adicionar or arquivos para o repositório local
2. git commit -m "Primeira versao do sistema"
3. git push
