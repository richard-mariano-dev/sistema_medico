# PRINCIPAIS COMANDOS DO GIT - Enviando a primeira versao
1. GIT INIT -> inicializar a pasta como repositório local
2. git branch -M main -> Altera a brach master para main
3. git add . -> Adicionar or arquivos para o repositório local
